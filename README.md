# DRL Cooperative Missile Guidance Tracker

Tracking Chinese research on Deep Reinforcement Learning for cooperative air-to-air missile guidance.

## Setup
```powershell
python -m venv venv
venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## Structure
- `collectors/` - Data collection scripts
- `analysis/` - Analysis and scoring
- `dashboard/` - Streamlit dashboard
- `data/` - Data storage
```
